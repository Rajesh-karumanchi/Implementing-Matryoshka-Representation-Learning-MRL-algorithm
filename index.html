Implementing Matryoshka Representation Learning (MRL) algorithm For CS 7150

What is MRL and How it helps in achieving flexibility in Representation Learning?

Proposal

The novel paper about Matryoksha Representation Learning is published in 2022. The idea of achieving flexibility for representation learning with less interaction from humankind, less computational resources and achieving same accuracy as the existing models grabbed our attention towards this project. For this project, as the paper is a novel one, we are planning to understand how Matryoksha Representation Learning algorithm works and implement the aspects covered in the paper to our fullest extent. We want to get a better understanding on the algorithm and interested in knowing how this model is better in using the computation resources by using coarse-to-fine representations. We will also try to evaluate MRL in different applications like Representation Learning, Adaptive Classification and Adaptive Retrieval.

Related Work

Y. Bengio [2] worked on using deep learning networks to perform representations for unsupervised learnings. There are many recent research papers on contrastive learning and enabled learning from large-scale web data. A lot of NLP applications are being built on large models are using masked language modelling or autoregressive training. In any neural network, we need efficient classification and retrieval using less space and cost. Y. A. Malkov and D. A. Yashunin[3] worked on finding an efficient and robust approximate nearest neighbor search using hierarchial graphs. The dependence of the complexity on the number of features effecting the computing costs and memory are studied in the above paper.

The main paper on MRL[1] can complement all the above designs, models and can easily adapt with minimal modification of existing representations and for no additional cost. MRL handles the dependency on the labels by lower the dimensions by taking the important features and learning multifidelity Matryoshka Representations. These lower dimensional representations are as accurate as the independently trained models with excessive costs and space usage.
